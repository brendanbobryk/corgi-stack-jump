A Stacker-inspired arcade React game where you help a corgi climb higher by stacking moving platforms perfectly!  
This project was built as part of a daily game challenge to practice React fundamentals, grid-based movement, overlap detection, and simple game loops.

## Features

- Grid-aligned, moving rows of blocks
- Click to drop the row onto the stack
- Only overlapping blocks survive, shrinking the row as you climb
- Increasing speed as the corgi stacks higher
- Dog mascot sits on top of the stack
- Score tracking for successful stacks
- Classic Stacker arcade feel
- Dark-themed, easy-on-the-eyes layout

## Technologies Used

- React (with hooks)
- JavaScript
- CSS

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/brendanbobryk/corgi-stack-jump.git

2. Navigate to the project folder:
    ```bash
    cd corgi-stack-jump

3. Install dependencies:
    ```bash
    npm install

4. Start the development server:
    ```bash
    npm run dev

5. Open your browser at the local address shown in the terminal (usually http://localhost:5173) to play!
